## Database Chat Engine
A chat engine that interacts with a SQL database using natural language queries, powered by OpenAI's GPT-4 and LangChain.

### Features
- Natural language to SQL query conversion
- Contextual understanding of database schema
- Safety measures to limit query results


### Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Shahjalal7311/ai-database-engine.git  
   ``` 


## Project setup
  - cd ai-database-engine
  - python -m venv venv
  - source venv/bin/activate
  - pip install -r requirements.txt
